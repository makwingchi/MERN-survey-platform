# Collec+

This is a web application developed using ***MERN*** stack. Users will be able to send out surveys via emails and also collect responses with the app.

## Stack
- `React`
- `React-Router`
- `Redux`
- `Redux-thunk`
- `Redux Form`
- `Node.js`
- `Express`
- `MongoDB`
- `Mongoose`
- `Google OAuth`
- `Passport.js`
- `SendGrid`
- `Stripe`
- `Semantic UI`
